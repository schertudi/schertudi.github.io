VRDR (Virtual Reality Design Review)

This project allows Enphase Energy to visualise the real-world usage of potential products. 3d models of designs can be imported into a session and viewed in the context of a typical customer's home. These models can be scaled, rotated, and moved around the house. Textures can be added to models and saved for later sessions. This can be run on both a standard desktop as well as VR, allowing users to get an accurate sense of product scale. Multiple users can join the same session to enable discussion of designs.

This was the conclusion to a project another intern had started the year before us. Essentially we needed to get the project from a prototype to a polished final product. I worked on this with another student over the span of the 2021-2022 summer break.

Before this I had done very little work in Unreal and had done nothing regarding networking, so it was a great learning experience for me.

My work

  • I worked on a system that allowed 3d objects to be customized with materials:
    • Different parts of the model can have different materials
    • Properties of the material can be customised through a GUI.
    • This was extended to allow these preferences to be stored and loaded from disk.
    • Configured objects also had to be synced from the host server to any clients, with any changes updated in realtime.
  • We also needed a system for loading 3d objects from a file on the network. I implemented this.
  • I also did general work on the program's various graphical interfaces, such as the menu to show control scheme
  • Configuration options eg input rebindings, configuration of session path
  • Documentation of final project to simplify work for any future developers
